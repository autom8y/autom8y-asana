# WS-P2-03: Protocol Purity + TYPE_CHECKING Guard Cleanup

**Objective**: Remove concrete implementation imports from `protocols/` files (making them
pure abstract interfaces), and remove TYPE_CHECKING guards that are no longer needed
after P2-02 cycle cuts.

**Rite**: hygiene
**Complexity**: PATCH
**Gaps**: G-05 (protocol purity), G-08 (guard cleanup)
**Preconditions**: P2-02 must be merged (G-08 depends on knowing which cycles were cut)
**Estimated Effort**: 2-3 hours

---

## Problem

**G-05 (Protocol Purity)**: Three files in `protocols/` import concrete types from
implementation modules. Protocols should define abstract interfaces using only stdlib
types, typing constructs, and other protocol types.

- `protocols/cache.py:10,14` -- imports from `cache.integration`
- `protocols/dataframe_provider.py:12` -- imports from `cache`
- `protocols/insights.py:15` -- imports from `clients`

**G-08 (Guard Cleanup)**: After P2-02 cuts 5 cycles, some TYPE_CHECKING guards that
existed to break those cycles are no longer necessary. Removing them simplifies the
code and makes imports explicit.

---

## Implementation Sketch

### G-05: Protocol Purity (1-1.5 hrs)

**Step 1**: Fix `protocols/cache.py`
1. Read `src/autom8_asana/protocols/cache.py` lines 1-30
2. Identify what is imported from `cache.integration` at lines 10, 14
3. Replace concrete type imports with either:
   - Protocol-level type aliases defined in the same file
   - TYPE_CHECKING guard (acceptable for protocols that need concrete return types)
   - Abstract type definitions using `typing.Protocol` or `typing.TypeVar`
4. Run: `pytest tests/unit/cache/ -x`

**Step 2**: Fix `protocols/dataframe_provider.py`
1. Read `src/autom8_asana/protocols/dataframe_provider.py` line 12
2. Identify the cache import
3. Replace with TYPE_CHECKING guard or abstract type
4. Run: `pytest tests/unit/query/ -x` (DataFrameProvider is used by query engine)

**Step 3**: Fix `protocols/insights.py`
1. Read `src/autom8_asana/protocols/insights.py` line 15
2. Identify the clients import
3. Replace with TYPE_CHECKING guard or abstract type
4. Run: `pytest tests/unit/models/ -x` (InsightsProvider is used by models)
