## Anti-Glossary (Ambiguous Terms to Avoid)

| Avoid | Use Instead |
|-------|-------------|
| "User" | Customer, Admin, Assignee (be specific) |
| "Data" | records, payload, entities (be specific) |
| "System" | Name the component |
| "Handle" | process, validate, transform |
